# Coin sound

Place a short coin/reward sound file here:

- **File name:** `coin.mp3`
- **Format:** MP3 (short clip, e.g. 0.3–1 second)

If `coin.mp3` is present, it will play when the user earns coins (e.g. daily check-in). If the file is missing, the app falls back to a system click sound.

You can use a free sound from [Freesound.org](https://freesound.org) (search for "coin" or "reward") and ensure the license allows use in your app.
